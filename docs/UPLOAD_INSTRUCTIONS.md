## Step 2: Extract and Upload Marketing Folder

The Marketing folder zip file should be extracted and its contents uploaded to:

```
public/assets/marketing/
```

### What to Include:
- All brand messaging documents
- Marketing strategy materials
- Course framework information
- Product descriptions
- Brand guidelines
- Any PDFs, Word docs, or other reference materials

### Organization Tips:
You may want to organize the marketing folder into subfolders like:
```
public/assets/marketing/
├── brand-guidelines/
├── messaging/
├── course-materials/
├── product-descriptions/
└── strategy/
```
