# Getting Started - The Faithful Mindset Repository

Welcome! This repository is ready for you to upload Hilary's Marketing materials and the Garden of Declarations book covers.

## Quick Start

### 1. Install Dependencies

```bash
cd the-faithful-mindset
npm install
```

This will install all required packages (Next.js, React, TailwindCSS, Framer Motion, etc.). The TypeScript errors you see now will disappear after installation.

### 2. Upload Your Materials

**Book Covers (12 images):**
- Location: `public/assets/book-covers/`
- See: `docs/UPLOAD_INSTRUCTIONS.md` for file naming

**Marketing Folder:**
- Location: `public/assets/marketing/`
- Extract the zip file and upload all contents

**Additional Images:**
- Location: `public/assets/images/`
- Any other visual assets

### 3. Run the Development Server

```bash
npm run dev
```

Visit [http://localhost:3030](http://localhost:3030) to see the site.

If the app shows missing chunk/module errors, run:

```bash
npm run clean:all
npm run dev
```
